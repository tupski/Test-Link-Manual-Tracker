📖 Panduan Test Link

🔗 Cara Test Link

  1. Buka aplikasi Test Link Tracker
  2. Pilih tab Test Link lalu pilih sesi (Pagi/Siang/Malam)
  3. Pilih kategori yang ingin di-test
  4. Tap tombol Buka pada setiap link
  5. Setelah link terbuka, pilih status: Normal ✅, Diblokir 🚫, atau Error ❌
  6. Ulangi untuk semua link dalam kategori
  7. Saat semua selesai, tap tombol 📤 Kirim Laporan

🧹 Cache & Cookies

Sebelum test link, bersihkan cache dan cookies browser agar hasil test akurat. Di Chrome: Pengaturan → Privasi → Hapus Data Penjelajahan.

📡 Provider & VPN

Gunakan provider asli (bukan VPN) saat test link agar hasilnya mencerminkan kondisi nyata. Lihat status VPN di bagian Kesiapan Test Link di beranda.
